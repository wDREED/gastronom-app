# 🍻 Store Manager MVP (Гастроном Пивники)

Легковесное десктопное приложение для автоматизации рутинных процессов кассира в розничном магазине разливных напитков и снеков. 

## 🎯 Какую проблему решает этот проект?

Изначально в магазине использовалась классическая бумажная тетрадь для записи PLU-кодов весовых товаров и отслеживания сроков годности. Это приводило к ряду бизнес-проблем:
* Тетрадь постоянно терялась или заливалась напитками.
* Появлялись дубликаты кодов, из-за чего возникала путаница на кассе.
* Отслеживание сроков годности фасованных снеков велось "в уме", что неизбежно приводило к человеческим ошибкам и просрочке на витринах.

**Решение:** Был разработан десктопный MVP (Minimum Viable Product), который оцифровал рабочее место сменщицы. Программа устанавливается на Windows-ПК кассы, не требует интернета и работает полностью автономно.

## 🚀 Основной функционал

* 📅 **Умный контроль сроков годности:** Система автоматически высчитывает остаток дней до истечения срока и маркирует позиции цветом (зеленый — в норме, желтый — осталось менее 14 дней, красный — просрочка). При запуске выдается алерт о критических товарах.
* ⚖️ **База PLU-кодов:** Мгновенный поиск товаров по названию или коду без необходимости листать справочники. Защита от добавления дубликатов.
* 💰 **Кассовый калькулятор:** Быстрый подсчет купюр по номиналам, автоматический расчет итоговой суммы в кассе и выручки с учетом остатка на размен.
* 📋 **Трекер задач смены:** Список дел (To-Do) с поддержкой drag-and-drop для изменения приоритета задач.
* 🗓 **Встроенный график:** Календарь рабочих смен с визуальным выделением рабочих и выходных дней.

## 🛠 Технологический стек и Архитектура

* **Язык:** Python 3
* **UI:** Tkinter (ttk)
* **База данных:** SQLite3 (встроенная)
* **Zero Dependencies:** Проект намеренно не использует сторонние библиотеки (отказ от изначальной кодогенерации штрихкодов `python-barcode` и `Pillow` в пользу легковесности и простоты развертывания).
